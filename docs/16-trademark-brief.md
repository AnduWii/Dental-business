# 16 · Trademark brief for "Catchline"

Practical plan for protecting the brand. This is operational guidance, not legal advice. Confirm
specifics and current fees with a registered trademark agent (Canada) or attorney (US), and on the
official `CIPO` and `USPTO` sites.

## The honest risk up front
"Catchline" is a real dictionary word (a catchy headline or opening line). Marks built from common
or descriptive words are harder to register and harder to enforce, because no one can monopolize an
ordinary word for its ordinary meaning. The upside: a common word used for unrelated goods or
services can still be protectable as a brand (the way "Apple" is for computers). For software and
messaging services, "Catchline" is arbitrary enough to have a real shot, but a clearance search is
essential before spending money, and a distinctive logo or stylized wordmark strengthens the claim.

## Step 1, clearance (do this before filing or printing business cards)
Search for existing conflicting marks, especially in software, telecom, and communications:
- CIPO Canadian Trademarks Database (the official register).
- USPTO search (if you want US rights).
- Plain web search, plus app stores and social handles.
- Domain availability for the name you intend to use commercially.
Look for identical or confusingly similar names in related classes. If you find a live mark for
software or communications services, talk to an agent before proceeding.

## Step 2, decide the owner
The mark should be owned by the company, Catchline Services Inc., once it is registered. Filing in a
personal name and assigning later is possible but adds paperwork. Register the company first if you
can (it is already on the pre-launch checklist).

## Step 3, pick the classes (Nice classification)
File in the classes that match what Catchline actually does:
- Class 9, downloadable software.
- Class 42, software as a service (SaaS), the core class for a hosted web app.
- Class 38, telecommunications and messaging services, if you position the SMS layer as a service
  you provide (worth discussing with an agent, since you resell carrier messaging).
For most SaaS, Class 9 and Class 42 are the backbone; add 38 on advice.

## Step 4, file
- Canada: file through CIPO's online system. A registered trademark agent is strongly recommended
  given the common-word issue.
- United States (optional, only if you will have US customers): file with the USPTO. If you are not
  yet using the mark in US commerce, an intent-to-use (1(b)) application holds your place.
- Consider filing the wordmark and, separately, a stylized logo mark. The logo is usually easier to
  register and gives you a second layer of protection.

## Step 5, costs and timing (approximate, verify before relying on them)
- Government filing fees: roughly a few hundred dollars per class in each of Canada and the US.
  Exact CIPO and USPTO fees change; check the official fee schedules.
- Agent or attorney fees vary widely (often several hundred to a couple thousand per filing).
- Timing is long: registration commonly takes from many months to well over a year in both
  countries, mostly examination queue time.

## What to do in the meantime
- You can use the "TM" symbol now to assert an unregistered claim. Use the "circle-R" symbol only
  after a mark actually registers.
- Lock down the matching domain and social handles now.
- Keep dated evidence of first commercial use (first paying clinic, first public launch).
- Keep the brand spelling and presentation consistent everywhere (see `docs/15-design-system.md`).

## Recommended sequence
1. Clearance search (cheap, fast, decisive).
2. If clear, register Catchline Services Inc.
3. Engage a Canadian trademark agent to file the wordmark and logo in Classes 9 and 42 (plus 38 on
   advice).
4. Add a US filing only if and when you pursue US customers.
